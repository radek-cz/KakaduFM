# KakaduFM

Another player for internet radio stations, using the database https://www.radio-browser.info/.

Requirements:
- Windows 10 (tested)
- Python 3.10+ https://www.python.org/downloads/

Installation:

    python -m venv venv
    pip install -r requirements.txt

Running:

    venv\Scripts\activate.bat
    python main.py
